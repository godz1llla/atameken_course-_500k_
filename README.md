# LMS System - Learning Management System

## Описание
Полнофункциональная система управления обучением (LMS) на чистом PHP без фреймворков.

## Технологии
- PHP 7.4+
- MySQL 5.7+
- HTML5, CSS3, JavaScript (Vanilla)

## Функционал

### Роли пользователей
1. **Администратор**
   - Управление пользователями
   - Управление курсами
   - Создание достижений
   - Назначение курсов студентам
   - Статистика

2. **Преподаватель**
   - Управление своими курсами
   - Просмотр студентов
   - Ответы на сообщения
   - Просмотр прогресса студентов

3. **Студент**
   - Прохождение курсов
   - Просмотр видео с YouTube
   - Прохождение тестов
   - Получение сертификатов
   - Сбор достижений
   - Общение с преподавателями

## Установка

### 1. Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Веб-сервер (Apache/Nginx)

### 2. Установка базы данных
```sql
mysql -u root -p < database/schema.sql
```

### 3. Настройка конфигурации
Отредактируйте `config/config.php`:
```php
define('BASE_URL', 'http://localhost');
define('DB_HOST', 'localhost');
define('DB_NAME', 'lms_db');
define('DB_USER', 'root');
define('DB_PASS', '');
```

### 4. Настройка прав доступа
```bash
chmod -R 755 public/uploads/
```

### 5. Доступ к системе
Перейдите по адресу: `http://localhost/index.php?url=auth/login`

**Админ аккаунт по умолчанию:**
- Email: admin@lms.com
- Password: password

## Основные возможности

### Система курсов
- Создание курсов с модулями и уроками
- Интеграция YouTube видео
- Текстовые материалы
- Отслеживание прогресса

### Система тестирования
- Создание тестов с вопросами
- Одиночный выбор ответа
- Ограничение по времени
- Ограничение попыток
- Проходной балл

### Достижения
- Автоматическая выдача при выполнении условий
- Типы достижений:
  - Завершение курса
  - Количество пройденных курсов
  - Высокий балл в тесте
  - Идеальный результат

### Сертификаты
- Автоматическая генерация при завершении курса
- Уникальный номер
- Возможность печати

### Система сообщений
- Личная переписка студент-преподаватель
- Уведомления о непрочитанных сообщениях
- История сообщений

## Безопасность
- Хеширование паролей (password_hash)
- Prepared statements для защиты от SQL-инъекций
- Разграничение прав доступа по ролям
- Валидация загружаемых файлов

## Использование без .htaccess
Система работает через параметры GET:
```
/index.php?url=controller/method/param
```

Примеры:
- Авторизация: `/index.php?url=auth/login`
- Дашборд студента: `/index.php?url=student/dashboard`
- Курс: `/index.php?url=student/course/1`

## YouTube интеграция
Вставьте полную ссылку на YouTube видео:
- https://www.youtube.com/watch?v=VIDEO_ID
- https://youtu.be/VIDEO_ID

Система автоматически создаст embed плеер.

## Разработка
Для добавления новых функций:
1. Создайте модель в `/models/`
2. Создайте контроллер в `/controllers/`
3. Создайте представление в `/views/`
